Мы считаем, что любые технологии, используемые на нашем сайте, должны быть опубликованы. Ниже представлен данный список.

Сервер

  • Основная часть сервера написана на Golang;
  • Фреймворк gin;
  • Для автогенерации документации используется swaggo;
  • Для работы с jwt-токенами используется jwt-go;
  • Для подключения к mysql базе данных используется go-mysql-driver.

Сайт

  • Для визуального оформления сайта используется Bootstrap;
  • Шрифты для сайта предоставлены Google Fonts.
